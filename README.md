# 🌐 NODUS Terminal v1.0
Система децентрализованной связи с интерфейсом в стиле киберпанк-терминала.

## ✨ Особенности
- **Real-time Messaging**: Мгновенный обмен сообщениями через WebSockets.
- **Node Switching**: Переключение между каналами (Общий, Metro Zone).
- **Call System**: Визуализация зашифрованных голосовых вызовов.
- **Adaptive UI**: Полностью рабочий интерфейс как на ПК, так и на мобильных устройствах.
- **Persistence**: Сохранение истории сообщений в SQLite.
