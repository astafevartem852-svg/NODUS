# Nodus OS | Ultimate Web Experience

Полноценная имитация операционной системы внутри мессенджера с интегрированным музыкальным сервисом и ИИ-ассистентом.

## ✨ Особенности
- **Nodus Music:** Полноценный стриминговый интерфейс с мини-плеером.
- **AI Integration:** Встроенный чат-бот Gemini.
- **Security:** Имитация 2FA и управления сессиями.
- **UI/UX:** Глубокий Blue-Black стиль, OLED Mode, плавные анимации и интро-загрузка.

## 🚀 Запуск
Просто откройте `index.html` в любом современном браузере.
